## 为什么排版在你的 Obsidian 库中很重要

在深入探讨特定搭配之前，理解字体在 Obsidian 中扮演的不同角色至关重要。Minimal 主题将排版分为三个主要部分：

1. **界面 (UI) 字体：** 这控制你的文件浏览器、设置菜单和命令面板。它需要紧凑，在小尺寸 (12px-14px) 下高度清晰，并且保持中性，这样就不会分散你对实际笔记的注意力。
2. **文本（正文）字体：** 这是你的库的核心。你的正文字体必须支持长时间阅读和写作。它需要出色的字距调整、独特的字母形状以防止视觉疲劳，以及舒适的 x 字高。通常设置在 15px 到 17px 之间。
3. **等宽字体：** 用于 Frontmatter、内联代码和代码块。它必须垂直对齐，清晰区分相似字符（如 `0` 和 `O`，或 `1`、`l` 和 `I`），理想情况下支持编程连字。

当这三个元素和谐共作时，你的库会感觉像是一个有凝聚力的高级应用程序。当它们冲突时，会引入微妙的、累积的摩擦，这可能会在长时间的工作中破坏你的专注力。

## 如何在 Obsidian Minimal 主题中配置字体

应用这些字体搭配需要几个简单的步骤。得益于 2026 年围绕 Minimal 主题的生态系统，你不需要编写自定义 CSS 就可以实现美观的设置。

1. **安装 Minimal Theme Settings 插件：** 这个配套插件是必需的。你可以在 Obsidian 社区插件目录中找到它。
2. **下载你的字体：** 确保你希望使用的字体已本地安装在你的操作系统上。Obsidian 会直接从系统的字体册中提取。
3. **配置插件：**
   - 进入 `Settings > Minimal Theme Settings`。
   - 向下滚动到 **Typography** 部分。
   - **Text Font：** 输入你的正文字体名称（例如 `iA Writer Quattro S`）。
   - **Interface Font：** 输入你的 UI 字体（例如 `Inter`）。
   - **Monospace Font：** 输入你的代码字体（例如 `Berkeley Mono`）。
4. **调整大小以实现和谐：** 不同的字体有不同的视觉尺寸。16px 的 Inter 看起来比 16px 的 Atkinson Hyperlegible 小得多。
   - 导航到 `Settings > Appearance`。
   - 调整 **Font Size** 滑块，直到正文感觉舒适（我们建议从 16px 开始）。
   - 使用 Minimal Theme Settings 插件独立调整 **UI Font Size**（通常在 13px 或 14px 最佳）。

### 高级提示：使用 CSS 为标题设置特定字体
如果你想只为你的 H1 和 H2 标题使用不同的字体以创造编辑上的对比，你可以添加一个简单的 CSS 代码片段：

```css
.markdown-rendered h1, 
.markdown-rendered h2 {
    font-family: 'Your Heading Font Name', sans-serif;
    font-weight: 700;
}
```
将此保存为你的 `.obsidian/snippets` 文件夹中的 `.css` 文件，并在“外观”设置中激活它。

## 优化排版以促进认知心流

选择字体只是成功的一半；该字体如何渲染决定了你的认知心流。在 Minimal 主题中配置你选择的搭配时，请牢记以下格式规则：

- **行长：** 阅读理解的理想行长在 60 到 80 个字符之间。在 Minimal Theme Settings 中，确保 **Normal line width** 设置在 40em 到 45em 左右。如果你的行长横跨整个显示器，当眼睛往回追踪下一行开头时会很快疲劳。
- **行高：** 如果你选择了具有高 x 字高的字体（如 Inter），你必须增加行高（行距），以防止文本看起来拥挤。对于 Obsidian 中的大多数无衬线字体，1.5 到 1.6 的行高是理想的。
- **对比度：** Minimal 允许你在高对比度和低对比度模式之间进行选择。如果你使用的是像 Lora 这样的细衬线字体，请选择较高对比度的文本，以确保极细的笔画不会在深色背景下显得模糊不清。

## 常见问题

### 我需要购买高级字体才能在 Obsidian 中获得良好的设置吗？
不需要。虽然像 Berkeley Mono 这样的高级字体提供了精致的细节和专门的字形，但像 Inter、IBM Plex 和 Roboto Flex 这样的开源字体是世界级的，完全能够以零成本在 Obsidian 中提供高级体验。

### 为什么我的字体在 Windows 上的 Obsidian 中看起来很模糊？
Windows 处理字体渲染 (ClearType) 的方式与 macOS 不同。如果你的字体看起来很细或模糊，请避免使用细体衬线字体。坚持使用像 Inter 或 Atkinson Hyperlegible 这样坚固的无衬线字体，并确保在 Windows 性能设置中启用了字体平滑。

### 我可以将我的自定义字体同步到 Obsidian Mobile 吗？
可以，但这需要额外的步骤。iOS 和 Android 不容易允许全系统范围的自定义字体安装。要在 Obsidian Mobile 上使用你选择的字体，你必须将字体文件转换为 Base64，并将其直接嵌入到你的库中的 CSS 代码片段里。

### 比例字体和等宽字体有什么区别？
在等宽字体中，每个字母占据完全相同的水平宽度（一个 'i' 和一个 'w' 一样宽）。在比例字体中，字母只占据它们需要的空间。等宽字体是编码和对齐的理想选择，而比例字体在快速阅读长段落方面则要好得多。

### 更改字体会影响 Obsidian 的性能或加载时间吗？
不会。只要你使用的是本地安装的系统字体，无论你在外观设置中指定哪种字体系列，Obsidian 都不会遭受任何性能下降。通过 CSS 嵌入的 Base64 字体可能会略微增加库的加载时间，但其影响通常可以忽略不计。
